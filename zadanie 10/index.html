1. Wymienić oraz opisz dwie części selektora,

    1. Selektor właściwy: Określa, do jakich elementów HTML zostanie zastosowana reguła stylu. Może to być np. h1, .przycisk, #naglowek.
    2. Blok deklaracji: Zawiera właściwości CSS i ich wartości, np. color: red; font-size: 16px;.

  2. Znaczenie nawisów klamrowym {...},

    3. Zawierają blok deklaracji, czyli listę właściwości CSS i ich wartości.Określają początek i koniec reguły stylu.

  3. Opisz cechy,

    4. Służą do precyzyjnego określania elementów HTML, do których ma zostać zastosowana reguła.Przykłady cech: id, class, type, href.

  4. Opisz wartości

    5. Określają parametry właściwości CSS, mogą to być np. kolory, rozmiary, czcionki, wartości liczbowe.

  5. Kiedy stosujemy znak średnika (;), kiedy nie jest konieczny

    6. Oddziela poszczególne deklaracje w bloku deklaracji.
