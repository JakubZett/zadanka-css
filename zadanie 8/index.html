1. gdzie jest wstawiany

    2. Wewnętrzny arkusz stylów jest wstawiany w sekcji nagłówkowej dokumentu HTML, pomiędzy znacznikami <head> i </head\>.

  2. składnia teoretyczna (w ramce, konieczne wcięcia)

    3. selektor {
      cecha1: wartość1;
      cecha2: wartość2;
    ... }

  3. jakie mogą być selektory,

    • Elementy HTML: np. h1, p, div
    • Klasy: .klasa
    • Identyfikatory: #identyfikator
    • Potomkowie: nadrzędny potomek
    • Rodzeństwo: poprzedni ~ kolejny
    • Pseudo-klasy: np. :hover, :focus

  4. jakie mogą być cechy i wartości.

    5. Cechy definiują właściwości elementu HTML, np. color, font-size, margin
    Wartości określają parametry danej cechy, np. #FF0000, 16px, 10px auto
